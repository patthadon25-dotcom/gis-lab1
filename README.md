## Project Description

This repository contains practice exercises for performing common GIS operations using Python libraries such as GeoPandas and Folium.

## Acknowledgements
This project is a practice implementation based on tutorials from the **GIS Applications** course.
The original tutorial uses **QGIS**, but this project reproduces similar workflows using **Python**.
